Titles

Every heading has an equivalent .title-[#] class that corresponds to its styles. This allows for HTML present to always be semantic, even if it needs to be rendered in a way that's tied to a different heading.

For example, if you needed an H3 tag to look like an H1, rather than putting an H1 where it didn't belong, you could use <h3 class="title-1">Title</h3> to achieve the desired look.
